# Vue Element Admin Example 

## 主要修改

- 添加 Vue ELement Admin 完整版的标签功能。
- 添加根据后台数据动态获取路由的方法。

## 运行命令

推荐使用 yarn 命令初始化项目

```sh
# 运行
yarn install

# 启动
yarn start
```

## 配置

- 页面功能配置：`src/settings.js`
- 默认路由配置：`src/router/modules/constantRoutes.js`
